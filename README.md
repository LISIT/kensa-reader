# vLLM-apps

家族みんなが、スマホで検査結果をやさしく読むための消費者向けアプリ群。

> ⚠️ **重要 / 非薬事・医療機器ではありません**
> ここに含まれるアプリは **医療機器ではなく**、診断・治療の判断に使うものではありません。
> 表示される内容はすべて「参考情報」であり、**必ず主治医・医療機関にご相談ください**。

## アプリ一覧

| アプリ | 概要 | 状態 |
|---|---|---|
| [kensa-reader](./kensa-reader/) | スマホ写真・スクショの **血液検査結果** をやさしい日本語で説明する PWA | 開発中 (v1) |

## 設計の柱

- **スマホで誰でも・インストール不要** — PWA。URLを開くだけ。
- **運用費ゼロ** — GitHub Pages 等の無料静的ホスティングで配布可能。
- **APIキー不要（既定）** — 既定エンジンはブラウザ内で完結。**画像は端末から外に出ません**。
- **任意アップグレード** — より自然な説明文が欲しい人は Claude API / ローカル Ollama を差し込み可能。
- **やさしいUI** — 大きな文字・3ステップ・色分け。高齢の家族でも使える。

詳細は [kensa-reader/README.md](./kensa-reader/README.md) を参照。
